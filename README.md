# Random Joke Generator 🎉

A simple Node.js application that fetches random jokes from the [JokeAPI](https://jokeapi.dev/).

## Features

- Fetches random jokes from an external API
- Supports both single-line and two-part jokes
- Easy to use and extend

## Installation

```bash
npm install
```

## Usage

Run the joke generator:

```bash
npm start
```

Or run it directly:

```bash
node joke-generator.js
```

## API

The application uses the [JokeAPI v2](https://v2.jokeapi.dev/) which provides:
- Single-line jokes
- Two-part jokes (setup and delivery)
- Multiple categories (General, Programming, Knock-knock, etc.)

## Example Output

```
Setup: Why did the programmer quit his job?
Delivery: Because he didn't get arrays.
```
